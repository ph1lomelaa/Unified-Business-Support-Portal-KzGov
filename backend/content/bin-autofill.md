## Первый шаг любой формы

Каждая форма заявки начинается со страницы «О компании» с полем БИН/ИИН (12
цифр). После ввода портал запрашивает профиль компании — в MVP это имитация
государственной базы данных юридических лиц (ГБД ЮЛ), в промышленном контуре
— реальный запрос к госорганам.

## Что заполняется автоматически

- Наименование компании
- Первый руководитель
- Регион регистрации
- ОКЭД (вид деятельности) — используется, в частности, для подбора мер
  поддержки по отрасли в навигаторе каталога

Эти поля становятся доступны только для чтения — их нельзя случайно
исправить вручную с ошибкой.

## Что автоматически заполняется в документах

В перечне документов встречаются пункты с пометкой «Получим по БИН» —
например, справка о государственной регистрации и справка об отсутствии
налоговой задолженности. Их не нужно прикладывать вручную — портал считает
их предоставленными автоматически на основании данных по БИН.

## Что нужно подготовить самому

Всё, что специфично для конкретного проекта, а не для юридического лица в
целом: бизнес-план, договор с поставщиком, финансовая отчётность, документы
по предмету лизинга. Такие пункты в списке документов помечены «Приложите
файл».
